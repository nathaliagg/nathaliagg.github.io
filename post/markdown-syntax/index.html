Markdown Syntax Quick Guide

Tue, Dec 29, 2020 2-minute read

These are the Markdown elements that I often used in Hugo content files, R markdown files, and Jupyter markdown cells. More markdown elements can be found here.

A - Headings

# H1
# H2
# H3
# H4
# H5
# H6

The output:

H1

H2

H3

H4

H5
H6

B - List Types

Ordered List

1. First item
2. Second item
3. Third item

The output:

  1. First item
  2. Second item
  3. Third item
Unordered List

* List item
* Another item
* And another item

The output:

  • List item
  • Another item
  • And another item

C - Tables

Name | Age
-----|----
 Bob | 27
Alice | 23

The output:

Name Age
Bob 27
Alice 23

D - Font Formatting

D.1 - Italic

*This text is italicized* or _This text is italicized_

The outputs:

This text is italicized or This text is italicized

D.2 - Bold

**This text is bold** or __This text is bold__

The outputs:

This text is bold or This text is bold

D.3 - Bold and Italic

***This text is really important*** or ___This text is really important___

The outputs:

This text is really important or This text is really important

E - Code Blocks

Python Example

``` <some code in here>
```

The output:

import pandas as pd

print('Hello World!')

Python Example with language spec

```python
<some python code in here>
```

The output:

import pandas as pd

print('Hello World!')

HTML Example

```html
<some html code in here>
```

The output:

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>

F - Adding links

I am loving this [Markdown](https://www.markdownguide.org/basic-syntax/) guide.

The output:

I am loving this Markdown guide.
